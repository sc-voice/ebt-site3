Welcome to EBT-Site

Site template for EBT-Vue3 applications

Overview

This is the site template for EBT-Vue3 applications.

To create your own EBT-Vue3 website, you'll need a Github account (Note: a free account is fine).

Create new repository

  1. Login to Github and navigate to <a href=https://github.com/sc-voice/ebt-site3" target="_blank">sc-voice/ebt-site3
  2. Then click the Use this template button, and choose Create new repository.

.

Configure new repository

After clicking Create new repository, you'll see a new webpage form for you to fill out:

  1. Specify the new repository name (e.g., "my-ebt-site")
  2. Choose Public repository (free)
  3. Select Include all branches
  4. Review your form entries
  5. When ready, click Create repository from template

We now need to enable Github pages for your new repository.

Enable Github pages

Navigate to your new repository and click Settings. Then click the "Pages" link in the left navigation bar. Configure the Build and deployment settings as shown below:

.

Once properly configured, you should see a link to your new website (e.g., ```https://my-account.github.io/my-ebt-site"). Follow the link to see your new website

Also see [Customize](/${baseName}/#/wiki/dev/customize} for customizing content.

Custom domains

Github supports custom domains.

The default URLs for Github pages are quite long. E.g.:

https://sc-voice.github.io/ebt-site3

It may therefore be worth the time and effort to have your own custom domain name. E.g.:

https://sc-voice.net

To get a custom domain name, you'll need to subscribe to a domain provider for a nominal fee. For example, SC-Voice uses Google Domains for the sc-voice.net internet domain name.
