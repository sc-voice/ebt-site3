Customize your EBT-Site

Create custom content using Markdown

Customize content

  1. Edit the content folder for your new repository as desired.
  2. Edit ebt.config.mjs and set appName
  3. Edit ebt.config.mjs and set github.account
  4. Edit ebt.config.mjs and set github.repository
  5. Edit index.html and set title
  6. Commit your new content
  7. View your new website

Custom content

Custom content is located in the content folder. Content webpages are generated from Markdown files.

For example, this web page is defined by:

content/welcome.md

Content channels

Organize your Markdown files into sub-folders of related content. Each content sub-directory is known as a channel.
